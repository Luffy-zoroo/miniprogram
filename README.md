用于无信号灯路口礼让行人的微信小程序开发
利用了微信开发者工具和腾讯云开发平台，通过JavaScript语言和微信小程序原生框架的代码，设计出了行人和驾驶人用户都可以使用并实现不同用户运动状态和实时位置数据的交互。
设计了行人界面，驾驶人界面和个人中心界面。
